Install Git in Linux :  
	1. yum install git -y 
Create Folder : 
	1. mkdir -p /home/hgorla/harish
	2. cd /home/hgorla/harish
Initialize Git in project Folder 
	1. git init
		a. Initialized empty Git repository in /home/hgorla/harish/.git/
	2. Check ls -a
		[hgorla@hint113 harish]$ ls -a
		.  ..  .git
		
	3. Now create Global Configurations
		a. git config --global user.name "hint113"
		b. git config --global user.email "hint113.com"
	4. Now check the configuration
		a. git config --list
	5. Create one file from your pc 
		a. vi day2.txt
	6. Check the status of the file 
		a. git status
